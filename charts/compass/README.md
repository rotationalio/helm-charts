# **DEPRECATED** — Django Compass

This Helm chart is deprecated and no longer recommended for new deployments.

Compass is now bundled with [Endeavor](https://github.com/rotationalio/endeavor) (embedded fixtures / `horizontasks`). Do not deploy `rotational/compass` for new environments.

Existing releases may remain until removed from the cluster; `helm install` and `helm upgrade` will print a deprecation warning.

# Image Tag (StarMap)

This chart wraps [Prodigy](https://prodi.gy/) with Rotational-specific use case to tag video stills from Google Cloud Storage.

This is not one of the official Rotational services, but is kept here for easy install into our clusters.

NOTE: we don't remember how the image in `gcr.io/rotationalio-habanero/imgtag:89ec287` was built, there is no associated repository. So far this is the only image that was generated for this tool.
